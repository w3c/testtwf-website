Come Join Us!

Test the Web Forward is throwing an event in Seattle on April 12–13. The focus of this event is on learning, hacking, and writing tests for W3C specifications. During the event, experts will teach you about W3C specs & W3C testing and will guide you as you help make the Web a better, more interoperable place. The Experts and Speakers will be posted shortly on the Test the Web Forward website. Also, follow us on @testthewebfwd for updates and be sure to sign up for the W3C Test the Web Forward mail list. To learn more about the past events, check out our posts on Test the Web Forward Paris, Beijing and San Francisco.

Get Registered

Registration is now open on Eventbrite, so get signed up! Help make the Web a better place!

Can't make it?

Hey, if you are busy or can't make it for any reason, it is a bummer, but be sure to sign up for the W3C Test the Web Forward mail list so that you learn about future happenings!
