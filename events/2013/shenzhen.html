Test the Web Forward将引领你更近距离接触你平时所用来创作Web体验的技术规范。在此次活动中，来自于W3C，Adobe，Intel，Mozilla， Google和Opera的专家们及众多独立开发者将手把手的引导你逐步学习阅读并理解相关规范，并且为这些规范创建和贡献全新的测试用例。你还可以了解到最新的HTML5和CSS3规范的最新状态并且获悉为何测试让浏览器一致实现这些新功能的迫切性。通过参加此次活动，你会正式投入一场塑造未来Web技术的运动中！。

Test the Web Forward brings you closer to the very specifications that you use to craft web experiences. At this event, experts from the W3C, Adobe, Intel, Mozilla, Google, and Opera as well as independent developers will be on hand to guide you step-by-step through reading & understanding specs and through creating & contributing new tests for them. You'll also learn the status of latest HTML5 and CSS3 specifications and why there is an urgent need for tests to make browser implementations of these features consistent. By participating in this event, you are joining a movement that is shaping the future of the web.
